# Design Adaptativo de Tema

> Memória do orbe — tópico `design-adaptativo-de-tema` (auto-gerada, filtrada de segredos).

## 2026-07-06
Foi decidido usar a variável CSS `--ink` para a cor do texto do subtítulo (`.brand .name span`). Isso garante que o texto se adapte automaticamente ao tema (branco no escuro e escuro no claro) para manter a legibilidade em ambos os modos.

# Estratégia de Cor para Textos

> Memória do orbe — tópico `estrategia-de-cor-para-textos` (auto-gerada, filtrada de segredos).

## 2026-07-06
Para textos na logo e subtítulo, é preferível usar a variável de tema `--ink` em vez de cores fixas. Isso garante que o texto seja legível (branco no tema escuro, escuro no tema claro) em diferentes modos de visualização, sem a necessidade de ajustes manuais.
